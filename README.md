# Biologically-Informed Algorithms for Modeling Cattle Response  
by Hannah James (Cornell) & Scott Baron (CSF)  

## Overview  
This repository hosts the code underlying the manuscript *“Biologically-informed algorithmic framework to simulate cattle responses to auditory cues in virtual fencing systems”*. The study integrates behavioural biology, animal-welfare science, and computational modelling to simulate how cattle (Bos taurus) respond to auditory cues in virtual fencing scenarios.  

Use-cases:  
- Reproduce all analyses, figures and tables published in the paper.  
- Apply the modelling framework to alternative cue-designs or herd settings.  
- Extend or adapt the algorithmic logic for other livestock‐oriented applications.

## Repository Contents  
```
/scripts/                - Python scripts implementing the simulation models  
/data/                   - (optional) sample input datasets, parameter tables  
/figures/                - output figures generated by the simulations  
/README.md               - this file  
/LICENSE                 - project licence  
```

## Computational Environment  
The analyses and simulations were conducted in **Google Colab** using:  
- Python 3.12.12 (GCC 11.4.0, glibc 2.35)  
- NumPy 2.0.2  
- Pandas 2.2.2  
- Matplotlib 3.10.0  
- SciPy 1.16.3  

You can reproduce the entire pipeline by running the notebook(s) or script(s) in the `/scripts/` folder.

## Reproducibility Instructions  
1. Clone or download this repository:  
   ```bash  
   git clone https://github.com/HannahJames123/Biologically-Informed-Algorithms-for-Modeling-Cattle-Response.git  
   ```  
2. Open the simulation notebook (or `main.py`) in Colab or your local environment.  
3. Install required dependencies (via `pip install numpy pandas matplotlib scipy`) or use the included `requirements.txt` if present.  
4. Place sample data files (if any) in `/data/`, then execute the code.  
5. Generated results (figures, tables) will appear in `/figures/` (or you can specify an output folder).  
6. Check parameter tables and allow for modification to test new scenarios.

## Parameter Documentation  
The key behavioural and simulation parameters (learning rate, habituation coefficient, response latency, etc.) are listed in the manuscript’s Supplementary Table S1 and in `/data/parameters.csv`. Each parameter includes:  
- Symbol used in equations  
- Empirical source or calibration method  
- Units or dimensionless designation
